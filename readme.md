# Arduino WebServer

Sketch para projetos de webserver com Arduino.

## WebServerBase
Sketch template para customização de webserver para o Arduino


## WebServerMonitor
Sketch codificado para automação remota:
1. Entradas Análogicas: Monitoramento de temperatura e umidade do ar
2. Entradas Digitais: Monitoramento de 5 entradas digitais
3. Saidas Digitais: Acionamento e monitoramento de 6 saidas digitais
